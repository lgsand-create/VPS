Anslut Gmail-konto

            

              
              

                1. Gmail API
                2. Auth Platform
                3. OAuth-klient
                4. Anslut
              


              
              

                
Steg 1: Skapa projekt & aktivera Gmail API

                
    
                  
  1. Gå till Google Cloud Console
    2. 
                  
  2. Skapa ett nytt projekt (eller välj befintligt) via projekt-väljaren högst upp
    3. 
                  
  3. Gå till APIs & Services → Enabled APIs & services
    4. 
                  
  4. Klicka + ENABLE APIS AND SERVICES, sök Gmail API och klicka Enable
    5. 
                

                

                  Tips: När Gmail API är aktiverat, gå till Google Auth Platform (meny → "Google Auth Platform") för att konfigurera OAuth.
                

                
              


              
              

                
Steg 2: Konfigurera Google Auth Platform

                
I menyn, gå till Google Auth Platform.

                
    
                  
  1. Branding — Fyll i appnamn: MailWise, din e-post som kontakt
    2. 
                  
  2. Audience — Välj External. Appen blir i "Testing"-läge
    3. 
                  
  3. Audience → Test users — Lägg till de Gmail-adresser du vill ansluta som testanvändare
    4. 
                  
  4. Data access — Klicka Add or remove scopes, sök gmail.readonly och bocka i den
    5. 
                

                

                  Viktigt: Alla Gmail-konton du vill ansluta måste läggas till som testanvändare så länge appen är i "Testing"-läge.
                

                
                
              


              
              

                
Steg 3: Skapa OAuth-klient

                
Fortfarande i Google Auth Platform:

                
    
                  
  1. Gå till Clients i vänstermenyn
    2. 
                  
  2. Klicka + CREATE CLIENT
    3. 
                  
  3. Välj typ: Web application
    4. 
                  
  4. Ge den ett namn (t.ex. MailWise VPS)
    5. 
                  
  5. Under Authorized redirect URIs, klicka + ADD URI och klistra in:
    6.

                  Efter att du skapat klienten visas Client ID och Client secret. Kopiera båda — du behöver dem i nästa steg. Du kan även hitta dem senare under Clients → klicka på klienten.
